``python3-openttd``
===================

A pure-python asyncio-based interface to the [admin network interface][1] of
[OpenTTD][2].

Requirements
------------

* [Python][0] 3.4+ (or Python 3.3 with tulip and enum34)

Basic usage
-----------

See the examples directory for examples :).
